# Superstore Sales Analysis using MySQL

## Project Overview
This project analyzes a retail Superstore sales dataset using SQL to extract meaningful business insights. The objective is to demonstrate strong SQL fundamentals, database design, and analytical thinking relevant to entry-level Data Analyst roles.

## Dataset
The dataset contains order-level transaction data including customer details, product information, sales, discounts, and profit.

## Tools Used
- MySQL
- MySQL Workbench

## Database Design
The raw dataset was first loaded into a staging table. It was then normalized into the following relational tables:
- customers
- products
- orders

Primary and foreign keys were used to maintain data integrity.

## Analysis Performed
- Total sales, profit, and number of unique orders
- Monthly sales trend analysis
- Regional sales and profitability analysis
- Top and bottom customers by revenue
- Impact of discounts on profitability
- Identification of loss-making products

## Key Skills Demonstrated
SQL, Joins, Aggregations, GROUP BY, HAVING, Data Cleaning, Database Design, Business Analysis

## Outcome
The analysis provides insights that can help stakeholders understand revenue trends, identify unprofitable products, and improve pricing and customer strategies.
